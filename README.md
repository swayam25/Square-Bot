## 🚀 Installation

1. Clone this repository
    ```sh
    git clone <repository url> square_bot
    cd square_bot
    ```

2. Install dependencies
    ```sh
    python -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    ```

3. Configure the [`config.json`](./configs/config.json) file

4. Start the bot
    ```sh
    source .venv/bin/activate
    python main.py
    ```
